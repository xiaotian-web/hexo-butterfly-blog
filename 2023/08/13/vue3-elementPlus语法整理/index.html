Typescript类型:

在 JavaScript 中，let 和 const 都是用来声明变量的关键字，它们的主要区别在于变量的可变性和作用域范围。

  • let 声明的变量是可变的（mutable），可以重新赋值，但不能重新声明。

  • const 声明的变量是不可变的（immutable），不能重新赋值，也不能重新声明。
